# Arch Linux 

## `pacman` 

**install package**

```shell
pacman -S <package name>
```

**update all installed packages**

```shell
pacman -Syu 
```
**Clear pacman cache**

```shell
sudo pacman -Scc
```

**uninstall package**

```shell
sudo pacman -Rs
```

**clean the cache in `~/home` directory

```shell
sudo du -sh ~/.cache/
`````
